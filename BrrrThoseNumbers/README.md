# Brrr Those Numbers

A Balatro mod that warns you when you're about to play your last hand, helping you avoid costly mistakes!

## Description

In the fast-paced world of Balatro, it's easy to get caught up in the excitement and accidentally play your final hand on economy boosts instead of maximizing your score. **Brrr Those Numbers** is here to save the day!

This mod displays a visual warning on the 'Play Hand' button whenever you have only one hand left, serving as a subtle but effective reminder to think twice before your next play. Stay focused on scoring big and avoid those "brrr" moments of regret.

### Features
- **Subtle Warning System**: Automatically detects when you have 1 hand remaining and displays a warning on the 'Play Hand' button.
- **Customizable**: Toggle the warning on or off via the in-game mod settings menu.
- **Non-Blocking**: Doesn't interrupt your gameplay, just provides a visual cue.
- **Seamless Integration**: Works with Steamodded framework, no conflicts with other mods.

2. Download the mod files from the [releases page](https://github.com/YoGeilmann/BrrrThoseNumbers/releases) or clone this repository.
3. Place the `BrrrThoseNumbers` folder into your Balatro mods directory (usually `%APPDATA%\Balatro\Mods` on Windows).

5. Access the mod settings via the Mods menu in-game to enable/disable the warning.

## Usage

- The 'Play Hand' button displays a warning when you have 1 hand left.
- The button returns to its normal color when the round ends or if you gain more hands.
- Customize the behavior in the mod's config tab within the game.

## Configuration

- **enable_warning**: Set to `true` to show warnings (default), `false` to disable.
- Edit `config.lua` manually or use the in-game settings menu.
