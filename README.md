# Brr Those Numbers

A mod for Balatro that helps prevent playing your last hand into economy instead of score.

## Description

This mod provides assistance to avoid common mistakes in Balatro where you might accidentally use your final hand on economy boosts rather than scoring.

## Installation

1. Download the mod files.
2. Place the `BurrThoseNumbers` folder into your Balatro mods directory.
3. Ensure the mod is enabled in the game.
